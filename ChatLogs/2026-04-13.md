## 플레이어 기능 복구 과정

### 이동
결과:
- WASD 이동 동작
- 카메라 기준 방향 이동 동작
- Idle / Running 애니메이션 동작

### 회피
문제:
- 처음엔 입력 자체가 안 먹거나 애니메이션과 상태가 어긋남
- 구르기 중 WASD에 흔들리는 문제 존재

해결 흐름:
- 입력 계층을 `DodgePressed`로 명시적 분리
- 애니메이션과 상태 지속 시간이 안 맞는 문제 발견
- `Roll.anim` 길이 확인: 약 `2.367초`
- 엘든링 기준 구르기 전체 길이 `0.6초`를 목표로 Animator `Dodge` 상태 속도 약 `4.0`으로 조정

핵심 인사이트:
- 구르기 애니메이션 길이와 `DodgeData.totalDuration`은 같은 개념이 아님
- 시각 재생 속도와 상태 판정 시간은 분리해서 다뤄야 함

추가 논의:
- 구르기 초반이 느리고 후반이 급하게 끝나는 체감
- 해결 방향:
  - `DodgeData.speedCurve`를 이용해 초반 가속 / 후반 감속 구조로 튜닝

### 약공격
결과:
- 좌클릭으로 `Slash1` 동작

현재 남은 문제:
- `Slash1 -> Slash2` 콤보는 아직 안정적이지 않음
- Animator 전이와 `AttackData.comboWindowStart / comboWindowEnd` 정렬이 필요

### 강공격
결과:
- `Shift + 좌클릭`으로 강공격 동작

입력 결정:
- 우클릭은 추후 패링용으로 비워두고
- 강공격은 `Shift + 좌클릭`으로 확정
